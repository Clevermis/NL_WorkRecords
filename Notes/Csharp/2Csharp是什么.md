## C#是什么
### C# 之所以能称为一门被广泛应用的编程语言，原因有以下几点：
	C# 是一种现代的通用的编程语言；
	C# 是面向对象的；
	C# 是面向组件的；
	C# 简单易学；
	C# 是一种结构化语言；
	使用 C# 开发效率很高；
	C# 可以在各种计算机平台上进行编译；
	C# 是 .Net Framework 的一部分。
	C# 强大的编程功能
	
### 以下是 C# 的一些重要功能的列表：
	布尔条件；
	自动垃圾回收；
	标准库；
	组件版本；
	属性和事件；
	委托和时间管理；
	易于使用的泛型；
	索引器；
	条件编译；
	简单的多线程；
	LINQ 和 Lambda 表达式；
	集成 Windows。

### 借助 C# 编程语言，我们可以开发不同类型且安全可靠的应用程序，例如：
	桌面应用程序；
	网络应用程序；
	分布式应用程序；
	Web 服务应用程序；
	数据库应用程序等。
